# 📈 Marketing CAC Analysis Dashboard

## 📌 Project Overview
Analyzed 500 marketing campaigns across 4 channels 
and 5 regions to identify cost-efficient customer 
acquisition strategies using MySQL and Power BI.

## 🛠️ Tools Used
- **MySQL** — Data storage and analysis
- **Power BI** — Interactive dashboard
- **Excel** — Data preparation

## 📊 Dataset
- 500 marketing campaigns
- 15 features including marketing channel, region,
  CAC, LTV, revenue, conversion rate and campaign status

## 🔍 Key Findings
- Overall Avg CAC: **$125.19** per customer
- **Referral** is cheapest channel ($119.89 CAC)
- **Email Marketing** is most expensive ($132.91 CAC)
- **Africa** has lowest CAC ($107.85) — most efficient
- **North America** has highest CAC ($134.55)
- **41%** campaigns are Unprofitable — needs attention
- **Premium Plan** generates highest revenue ($704,485)
- CAC improved over time — Dec 2025 lower than Apr 2024

## 💡 SQL Analysis Covered
- Overall CAC and marketing spend
- CAC by marketing channel
- CAC by region
- Campaign status analysis
- CAC by product category
- Monthly CAC trend
